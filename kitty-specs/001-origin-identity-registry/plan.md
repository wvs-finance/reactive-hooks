# Implementation Plan: Origin Identity Registry

**Branch**: `feat/subscription-lib-replace-mod` | **Date**: 2026-03-16 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `kitty-specs/001-origin-identity-registry/spec.md`

## Summary

Implement the foundation type system and registry for origin endpoints in the Event Dispatch Table (EDT). Origins are cross-chain event sources identified by a deterministic `keccak256(chainId, emitter, eventSig)` hash. The registry provides O(1) lookup by hash and enumeration by chainId via dual-representation storage, following the project's SCOP/Compose conventions (file-level free functions, namespaced storage slots, no inheritance or library keyword).

## Technical Context

**Language/Version**: Solidity 0.8.26, EVM target Cancun
**Primary Dependencies**: reactive-lib, solady, Compose, forge-std
**Storage**: Namespaced storage slots (keccak256 slot isolation), dual-representation (hash map + per-chain list)
**Testing**: Foundry (forge test), Kontrol (formal verification with prove_ prefix)
**Target Platform**: Reactive Network (dual-instance: RN + ReactVM)
**Project Type**: Smart contract library (Foundry project)
**Performance Goals**: O(1) origin lookup by hash, O(n) enumeration per chain
**Constraints**: SCOP — no library keyword, no inheritance, no modifiers. File-level free functions only.
**Scale/Scope**: ~10 origins per chain typical, ~100 max. Gas-optimized for writes (registration is infrequent, lookups are hot path).

## Phase 1: Design

### Data Model

**OriginEndpoint** (value type, no storage):
- `chainId`: uint32 — chain where the event is emitted
- `emitter`: address — contract emitting the event
- `eventSig`: bytes32 — topic_0 (event signature hash)

**originId** (derived, deterministic):
- `keccak256(abi.encodePacked(chainId, emitter, eventSig))` → bytes32

**OriginRegistryStorage** (namespaced storage struct):
- `origins`: mapping(bytes32 => OriginEndpoint) — hash → struct
- `exists`: mapping(bytes32 => bool) — dedup guard
- `originsByChain`: mapping(uint32 => bytes32[]) — chain → list of originIds
- `totalCount`: uint256 — total origins across all chains
